# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Professional Python packaging with pyproject.toml
- Comprehensive CI/CD pipeline with GitHub Actions
- Pre-commit hooks for code quality
- MIT License
- EditorConfig for consistent code formatting
- Type hints and docstrings throughout codebase
- Comprehensive test suite with pytest
- Code quality tools (black, isort, flake8, mypy)
- Security scanning with safety and bandit
- Multi-platform testing (Ubuntu, macOS, Windows)

### Changed
- Modernized project structure following Python best practices
- Improved documentation organization
- Enhanced README with badges and comprehensive examples

### Fixed
- Code formatting standardized with black
- Import sorting with isort
- Type checking with mypy

## [0.1.0] - 2024-11-10

### Added
- Initial release of forecasting framework
- Core forecasting infrastructure (ForecastData, ForecastResult, BaseForecaster)
- Time series models (ARIMA, Prophet)
- Econometric models (VAR, VECM, State Space)
- Machine learning models
- Bayesian forecasting methods
- Ensemble methods
- Evaluation metrics (Brier Score, Log Score, CRPS)
- Statistical tests (Diebold-Mariano, Ljung-Box, ADF)
- Calibration tools (Isotonic Regression, Platt Scaling, Temperature Scaling)
- Uncertainty quantification
- Visualization tools
- Competition data loaders (Metaculus, GJ Open)
- Economic data loaders (World Bank API)
- Comprehensive documentation and examples
